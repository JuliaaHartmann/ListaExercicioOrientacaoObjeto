# Lista Exercícios de Orientação a Objetos Básica

## Projeto

Desenvolvido durante o curso Fullstack da [Academia do Programador](https://www.academiadoprogramador.net) 2026

## Introdução

Este projeto reúne uma série de exercícios práticos de Orientação a Objetos com foco em conceitos fundamentais de programação. Cada exercício propõe a criação de pequenos programas que exploram cálculos matemáticos, conversões de unidades e manipulação de dados, permitindo ao estudante aplicar os princípios básicos de classes, métodos e atributos.

Os exercícios incluem:

- Cálculo de volumes (caixa retangular, cilindro, esfera, lata de óleo).
- Conversão de temperaturas entre Celsius e Fahrenheit.
- Cálculo de consumo de combustível por quilômetro.
- Cálculo de salário total de um vendedor com comissão.
- Cálculo de médias (harmônica e ponderada).

Essas atividades têm como objetivo consolidar o aprendizado de lógica de programação e introduzir boas práticas de desenvolvimento orientado a objetos.

## Como utilizar

1. Clone o repositório ou baixe o código fonte.
2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz
3. Utilize o comando abaixo para restaurar as dependências do projeto.

   ```bash
   dotnet restore
   ```

4. Para executar um exercício específico, utilize o comando abaixo substituindo pelo nome do projeto desejado:

   ```bash
   dotnet run --project Exercicio01.ConsoleApp
   ```
Exemplos:

- Executar o exercício 1 (volume da caixa retangular):
    dotnet run --project Exercicio01.ConsoleApp

- Executar o exercício 2 (conversão Fahrenheit → Celsius):
    dotnet run --project Exercicio02.ConsoleApp

- Executar o exercício 3 (volume do cilindro):
    dotnet run --project Exercicio03.ConsoleApp

## Requisitos

- .NET 10.0 SDK
